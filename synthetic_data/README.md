# Synthetic Data
Synthetic CSV rows used to instantiate APLO for validation and competency question examples.

Each CSV corresponds to a matching YAML mapping in [foundry_mappings/synthetic_sample/](../foundry_mappings/synthetic_sample/) and a matching generated RDF folder in [materialization/](../materialization/).

| Group | Files |
| --- | --- |
| Pedagogical structure | `pedagogical_frameworks.csv`, `pedagogical_elements.csv`, `pedagogical_techniques.csv` |
| Learning objectives and resources | `learning_objectives.csv`, `learning_situations.csv`, `learning_strategies.csv`, `modules.csv`, `media.csv`, `skills.csv` |
| Learner model | `learners.csv`, `learner_goals.csv`, `learning_challenges.csv`, `awareness_observations.csv` |
| Evaluation and feedback | `evaluation_assessment_exercises.csv`, `evaluation_techniques.csv`, `evaluation_metrics.csv`, `evaluation_results.csv`, `learning_feedback.csv`, `responses.csv`, `sources.csv` |

# Autonomous Learning Orchestration Ontology
This repository contains an ontology and supporting artifacts for modeling pedagogical frameworks, learning objectives, learner state, and evaluation concepts.

## Repository Layout

- `schema/`
  - Canonical ontology artifacts and visual exports.
  - Use the `.graphml` files for diagram editing, and the `.pdf` files for quick review.
- `documentation/`
  - Human-readable context for the ontology.
  - `key-notions.md` explains the main modeled concepts.
  - `competency-questions.md` captures the questions the ontology is intended to answer.
- `axioms/`
  - Constraint and axiom definitions in tabular form.
  - `axioms.csv` is the main reference for relationships, cardinalities, and rule rationale.
